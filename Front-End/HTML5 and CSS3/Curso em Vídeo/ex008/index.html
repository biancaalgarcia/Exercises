Principais Formatações

Negrito/Destaque

Nessa frase temos uma tag não semântica em negritousando a tag <b>

Nessa frase temos uma tag semântica em negritousando a tag <strong>

Itálico / Ênfase

Nessa frase temos uma tag não semântica em itálicousando a tag <i>

Nessa frase temos uma tag semântica em itálico usando a tag <em>

Texto marcado

Podemos ter uma tag semântica marcada usando a tag <mark>

Texto grande e pequeno

Podemos usar a tag semântica <big>(obsoleta) para textos grandes e <small>(não obsoleto) para textos pequenos

Texto deletado

Usamos a tag <del> para textos riscados(tag semântica)

Texto inserido

Usamos a tag <ins> para textos sublinhados para dar ênfase(tag semântica)

Usamos também a tag <u> para textos sublinhados(tag não semântica)para dar ênfase

Para textos sobrescritos

Use a tag<sup>, exemplo: x20 + 5

Para textos subscritos

Use a tag <sub>, exemplo: H2O
